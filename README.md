# yieldCurvesGlobal
Global Bond Yield Curves - US, UK, GER, FRA  0Y-30Y

Product description 

https://yieldcurvesglobal.up.railway.app

Real-time Data
I get real-time data for US, UK, GER, FRA (though the FinanceFlowAPI source charges me per API call so I default the date picker to previous day).🤣

Historical Data
St Louis Fed has a FRED API for all historical data for USTs. 
This you can easily view. At the moment they don’t have data for March 11, 2026. But they have March 10, 2026 and earlier. So you can view this data across the curve or you can change country mode to tenor mode and view historic data for each point on the curve , eg: 10Y or 2Y.
